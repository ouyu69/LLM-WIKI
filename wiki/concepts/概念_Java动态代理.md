# 概念：Java 动态代理

## 定义
代理模式通过代理对象控制对目标对象的访问。动态代理是在运行期生成代理对象，相比静态代理减少手写代理类。

## JDK 动态代理
- 基于接口生成代理类。
- 核心 API 是 `Proxy` 与 `InvocationHandler`。
- 适合目标对象实现了接口的场景。

## CGLIB 动态代理
- 通过继承目标类并生成子类实现代理。
- 不要求目标类实现接口。
- 不能代理 `final` 类或 `final` 方法。

## 应用场景
- Spring AOP、事务、日志、权限校验、RPC 调用包装。
- 在方法调用前后统一添加横切逻辑，减少业务代码重复。
