# 概念：值传递与引用传递

## 来源中的定义
- 值传递：实参复制给形参，形参变化不影响实参。
- 引用传递：传递地址，函数内部修改会影响外部对象。

## Java 语义澄清
- Java 只有值传递。
- 当参数是对象时，传递的是“引用的值”，因此可通过该引用修改对象内部状态，但无法让调用方变量指向新对象。
