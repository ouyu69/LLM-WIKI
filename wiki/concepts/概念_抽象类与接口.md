# 概念：抽象类与接口

## 共同点
- 都不能直接实例化。
- 都可描述抽象行为。

## 差异
| 维度 | 抽象类 | 接口 |
|---|---|---|
| 定义 | `abstract class` | `interface` |
| 继承关系 | 单继承 | 可多实现 |
| 构造器 | 可定义 | 不可定义 |
| 字段 | 可定义实例字段 | 字段默认 `public static final` |
| 方法 | 抽象 + 具体方法 | Java 8+ 支持默认/静态方法 |

## 选择建议
- 需要共享状态或模板实现：优先抽象类。
- 需要定义跨类型契约：优先接口。
