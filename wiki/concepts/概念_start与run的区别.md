# 概念：start 与 run 的区别

## `start()`
- 触发线程启动流程。
- 线程进入可运行态，由调度器分配时间片后执行 `run()`。

## `run()`
- 直接调用时只是当前线程中的普通方法。
- 不会产生新的执行线程。
