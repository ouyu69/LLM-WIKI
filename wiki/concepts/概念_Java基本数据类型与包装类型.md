# 概念：Java 基本数据类型与包装类型

## 定义
Java 基本数据类型包括 `byte`、`short`、`int`、`long`、`float`、`double`、`char`、`boolean`。每种基本类型都有对应包装类，例如 `Integer`、`Long`、`Boolean`。

## 核心差异
- 用途：基本类型适合简单数值和局部计算；包装类型可用于泛型、集合、反射和需要对象语义的场景。
- 默认值：成员变量基本类型有默认值，包装类型默认是 `null`。
- 比较方式：基本类型用 `==` 比较值；包装类型的 `==` 比较引用，值比较应使用 `equals()`。
- 存储与开销：包装类型是对象，通常有额外对象头和引用开销。
- 缓存机制：部分包装类会缓存常用值，例如 `Integer` 默认缓存 `-128` 到 `127`，因此不能用 `==` 推断所有包装对象值相等。

## 数值精度
- 浮点数使用二进制表示小数，部分十进制小数无法精确表示，因此可能出现精度误差。
- 金额和高精度计算通常优先使用 `BigDecimal`，并显式指定舍入规则。
- 超过 `long` 范围的整数可使用 `BigInteger`。
