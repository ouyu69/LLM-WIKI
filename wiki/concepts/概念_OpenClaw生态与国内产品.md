# 概念：OpenClaw 生态与国内产品

## 社区文化
OpenClaw 的吉祥物是龙虾，中文社区将运行 OpenClaw 称为“养虾”，用户自称“养虾人”。这种文化标签降低了技术项目的传播门槛。

## 生态形态
- ClawHub Skills：围绕 Agent 能力扩展形成技能市场。
- Moltbook：来源文档描述的 AI Agent 社交网络，供 Agent 发帖、评论和互动。
- 国内教程：B站、知乎、博客园、云厂商文档围绕安装、部署、飞书/钉钉/QQ 接入形成大量内容。
- openclaw-china：文档提到的国内 IM 适配插件，覆盖钉钉、QQ、企业微信、飞书等。

## 国产产品阵营
- OpenClaw 封装版：QClaw、MaxClaw、KimiClaw、AutoClaw、ArkClaw 等，通常降低部署门槛但可能绑定模型或落后上游。
- 独立自研版：miclaw、LobsterAI、CoPaw 等，不一定兼容 ClawHub，但可能在本土生态或设备联动上更深入。
- 兼容生态产品：WorkBuddy 等强调兼容 OpenClaw 技能生态。

## 选型提醒
如果目标是学习和深度折腾，原版 OpenClaw 最灵活；如果目标是快速体验，封装版更省事；如果关注数据控制和可审计性，优先选择开源且可本地部署的方案。
