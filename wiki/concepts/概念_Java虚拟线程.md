# 概念：Java 虚拟线程

## 定义
虚拟线程是 Java 21 正式发布的轻量级线程能力，目标是以接近传统同步代码的写法承载大量并发阻塞任务。

## 适用场景
- 高并发网络 IO。
- 大量短生命周期任务。
- 需要保留同步编程模型但希望降低平台线程占用的服务端程序。

## 不适用场景
- CPU 密集型计算：吞吐仍受 CPU 核心数约束。
- 长时间持有锁或阻塞在不兼容的本地调用中：可能削弱虚拟线程优势。
- 需要严格控制有限外部资源时：仍然需要连接池、限流和背压。

## 面试记忆点
虚拟线程降低的是线程创建和阻塞等待成本，不是让单机 CPU 算力变多。它让“一个请求一个线程”的模型重新变得可行，但资源治理仍然必须存在。
