# 概念：Java I/O 模型

## IO 流
I/O 表示输入和输出。Java I/O 流按数据单位可分为字节流和字符流。

## 字节流与字符流
- 字节流以字节为单位，适合图片、音频、二进制文件等任意数据。
- 字符流以字符为单位，适合文本数据，会涉及字符编码与解码。

## 设计模式
Java I/O 中大量使用装饰器模式，例如通过缓冲流为基础流增强缓冲能力；也使用适配器思想在字节流和字符流之间转换。

## BIO、NIO、AIO
- BIO：同步阻塞 I/O，一个连接通常对应一个线程，简单但高并发成本高。
- NIO：同步非阻塞 I/O，基于 Channel、Buffer、Selector，可用较少线程管理大量连接。
- AIO：异步 I/O，操作完成后通过回调或通知处理结果。
