# 概念：线程与进程

## 关系
- 进程是程序运行的基本单位。
- 线程是进程内的执行单元，一个进程可含多个线程。

## 资源视角
- 同进程线程共享：堆、方法区（或元空间）。
- 线程私有：程序计数器、虚拟机栈、本地方法栈。

## Java 与操作系统
- 现代 Java 线程基于原生线程实现。
- 主流操作系统上通常采用一对一线程模型。
